# Food Delivery Data Analysis - Innomatics Hackathon

This project is a data analysis study on a food delivery system using Python and Jupyter Notebook.  
It explores customer orders, restaurants, and user data to gain useful insights.

---

# Project Structure

Innomatics_Hacathon/
│
├── food_delivery_analysis1.ipynb # Main analysis notebook
├── final_food_delivery_dataset.csv # Processed dataset
├── orders.csv # Orders data
├── restaurants.sql # Restaurant database
├── users.json # User information
└── README.md # Project documentation

## Objectives

- Analyze food delivery data
- Understand customer ordering patterns
- Study restaurant performance
- Clean and preprocess datasets
- Visualize insights using Python
